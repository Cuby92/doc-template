RP2350

Jakub Barczyński

1. Architecture

1.1. Specification

  • Dual-core Cortex-M33 or Hazard3 operating at 150MHz
  • 520kB embedded SRAM memory
  • 8kB one-time programmable (OTP) memory
  • Built-in switched-mode power supply
  • Optional low LDO mode with low standby current for sleep states
  • 2 built-in PLL loops for internal or external clock signal generation
  • GPIO pins tolerate 5V (under power) and have failsafe 3.3V protection (unpowered)

1.2. Peripherals

  • 2 UART interfaces
  • 2 SPI controllers
  • 2 I2C controllers
  • 24 PWM channels
  • USB 1.1 controller and PHY with support for host and device modes (interface providing Full Speed (FS) and Low Speed (LS) connectivity under software control)
  • 12 PIO state machines (these machines are grouped in three programmable input/output (PIO) blocks, each containing four independent units capable of handling various communication standards)
  • HSTX peripheral (a block designed for high-speed data transmission that can be assigned to GPIO pins)

2. Instruction Set

  • JMP - Jump to a specified address.
  • WAIT - Suspend program execution until a specified condition is met on a GPIO pin, IRQ flag, or dedicated JMP pin.
  • IN - Shift a specified number of bits (1–32) from a source (pins, X, Y, ISR, OSR registers) into the input ISR register.
  • OUT - Shift bits from the output OSR register to a destination.
  • PUSH - Send the contents of the ISR register to the RX FIFO queue as a 32-bit word and clear ISR
  • MOV - Copy data between registers. Can perform negation or bit-reversal operations
  • PULL - Fetch a 32-bit word from the TX FIFO queue to the OSR register
  • IRQ - Set, clear, or wait for an interrupt flag
  • SET - Directly write a 5-bit immediate value to pins or registers

4. Microcontroller Variants

4.1. RP2350A

  • Package: QNF-60 (7x7 mm)
  • GPIO Pin Count: 30
  • Analog Inputs: 4

4.2. RP2350B

  • Package: QFN-80 (10x10 mm)
  • GPIO Pin Count: 48
  • Analog Inputs: 8
